
# Letsbloom library API 
A simple RESTful API for managing a library system using Node.js and mongodb.






## Run Locally

Clone the project

```bash
  git clone https://github.com/Krishhh18/library.git
```

Go to the project directory

```bash
  cd library
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node app.js
